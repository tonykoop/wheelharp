# Assembly Sequence

This sequence is an L3-candidate planning order, not a fabrication release. It
names build logic, jigs, and hold points without dimensions or released
geometry.

## Suggested Build Order

1. Build a one-wheel witness rig.
2. Add a single string and adjustable contact fixture.
3. Add a removable damper and verify note-off behavior.
4. Add a key or lever coupon to control engagement.
5. Add a guarded drive source and observe heat, noise, and dust.
6. Repeat service operations: clean wheel, re-rosin, replace string, adjust
   damper, inspect guard.
7. Only after the single-note chain is stable, mock up a small multi-note row.
8. Only after multi-note interaction is understood, review semicircular
   keyboard routing.

## Jigs And Fixtures

| fixture | role | authority |
| --- | --- | --- |
| one-wheel witness rig | isolates wheel truing, rosin, and contact behavior | pending_measurement |
| string contact fixture | lets contact path vary without claiming final action geometry | pending_measurement |
| damper test fixture | compares note-off behavior without fixed spacing claims | pending_measurement |
| key action coupon | evaluates travel, return, and lost motion | pending_measurement |
| drive isolation stand | separates motor noise from wheel/string tone | pending_measurement |
| guard mockup | reviews safety and cleaning access | pending_measurement |
| service-access mockup | checks removal and adjustment sequence | pending_measurement |

## Assembly Hold Points

- Stop after wheel truing if runout or wobble cannot be inspected repeatedly.
- Stop after first bow contact if rosin or string prep is not logged.
- Stop after damper install if note-off introduces rebound or scraping.
- Stop before powered tests if guards and electrical isolation are not reviewed.
- Stop before multi-note scaling if service operations disturb alignment.

## Disassembly And Service Checks

Every prototype stage must be reversible enough to answer:

- Can the wheel be cleaned and inspected?
- Can the string be replaced?
- Can the damper be adjusted?
- Can the guard be removed and reinstalled without losing alignment?
- Can rosin dust be removed without contacting electrical parts?
